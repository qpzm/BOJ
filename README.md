## Javascript

### Jest

Run `yarn test PATTERN`.
- `--watch` runs tests automatically on file change.
- Specify tests to run with regex. The suffix of tests should be `.test.js` not `_test.js`.

### Typescript

Run `tsc -w PATTERN` to compile on file change.
